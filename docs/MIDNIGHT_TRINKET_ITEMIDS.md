# Midnight Trinket Item IDs

WoWHead search results 기준으로 2026-03-17에 확인한 한밤 장신구 `itemId` 목록이다.

주의:

- 이번 확인은 WoWHead 검색 결과 페이지에서 `item=...` 패턴을 추출하는 방식으로 수집했다.
- 실제 드랍 난이도별 `bonus_id`, `ilevel`은 별도 확인이 필요하다.
- 이름 토큰과 함께 `itemId`를 병행 유지하면 simc 호환성 점검에 유리하다.
